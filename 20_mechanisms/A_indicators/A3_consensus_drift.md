## 1. Measurement Definition

### What We're Tracking
The **Consensus Drift** indicator monitors:
- Movement toward or away from value convergence
- Rigidity or flexibility of moral consensus
- Trend in tolerance for dissent and alternative values
- Hardening or softening of institutional positions on core values

### The Core Metric
```
Consensus_Rigidity = (Strength_of_Consensus / Institutional_Flexibility) Ã— 
                     (Penalty_for_Dissent / Space_for_Alternative_Values)
```

**Assessment:**
- **Rigidity < 0.5** = Consensus is loose; flexibility high; values remain adaptive (NOMINAL)
- **Rigidity 0.5-1.0** = Consensus tightening; flexibility declining (CAUTION)
- **Rigidity > 1.0** = Consensus is iron; flexibility minimal; lock-in underway (CRITICAL)

### Why This Matters
Lock-in doesn't happen all at once. It emerges gradually as consensus hardens and flexibility declines. Early detection of consensus rigidity is essential for preventing value lock-in before it's irreversible.

## 4. Interpretation Guidance

### Low Rigidity (NOMINAL)
- **Meaning**: Consensus is soft; flexibility is high; values remain changeable
- **Implication**: Lock-in risk is low; civilization can still evolve values
- **Action**: Maintain institutions that support value flexibility and diversity

### Moderate Rigidity (CAUTION)
- **Meaning**: Consensus is tightening; flexibility declining; rigidity emerging
- **Implication**: Lock-in risk is increasing; time to act is limited
- **Actions**: 
  - Increase protection for moral dissent
  - Strengthen institutions supporting value diversity
  - Reduce penalties for holding alternative values
  - Create space for moral debate
  - Protect minority value systems
  - Slow institutional consolidation around single values

### High Rigidity (CRITICAL)
- **Meaning**: Consensus has hardened; flexibility minimal; lock-in underway or complete
- **Implication**: Civilization has lost ability to change core values
- **Actions**: Trigger A3 emergency protocols
  - Immediately reverse penalties for dissent
  - Revive suppressed moral alternative systems
  - Break institutional enforcement of consensus
  - Restore institutional flexibility
  - Create institutional mechanisms for value reconsideration
  - Restore debate and dialogue space

## 5. Failure Modes & Limitations

### How This Indicator Could Be Misread

**False Positive Risk:**
- *Rhetoric vs. reality*: Institutions claim openness to moral debate while suppressing it
- *Generational confusion*: Different generational values appearing as consensus shift when it's just demographic change
- *Temporary volatility*: Short-term fluctuations in polling appearing as genuine value change

**False Negative Risk:**
- *Invisible consensus*: Consensus hardening beneath surface of apparent pluralism
- *Slow lock-in*: Rigidity increasing so gradually it goes unnoticed
- *Undetected suppression*: Dissent being quietly punished without visible conflict

### How to Mitigate
1. **Track actual dissent penalties**: Don't assume tolerance exists; document actual consequences
2. **Monitor longitudinal trends**: Look for direction of change, not just snapshots
3. **Test institutional willingness**: When values are actually questioned, institutions reveal their true rigidity
4. **Watch generational succession**: Can younger generation reconsider parents' values, or is it locked?
5. **Monitor late-stage signals**: As rigidity increases, watch for decreasing tolerance for dissent
