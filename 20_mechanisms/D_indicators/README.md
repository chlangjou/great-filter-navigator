# D Indicators: Infrastructure Resilience and Orbital Security

These indicators monitor whether civilization is losing the physical infrastructure required for global coordination, orbital awareness, and outward expansion.

## Implemented Indicators

- **D1.1 Orbital Debris Cascade Index**: Tracks whether LEO congestion and debris dynamics are approaching a Kessler-style lockout.
- **D1.2 Global Network Fragmentation Index**: Tracks whether satellite loss, cable disruption, and information-sovereignty policies are breaking the global Internet into coercible regional intranets.
